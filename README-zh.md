# obs-auto-subtitle
OBS自动字幕插件，一边讲话一边自动生成字幕！

目前仅支持普通话

[English](./README.md)

## Feature
- **自动字幕过滤器** : 转码音频并发送到云平台。
- **阿里云 NLS**: 支持使用阿里云 NLS 服务进行字幕生成。
- **讯飞AI云 RTASR**: 支持使用讯飞 AI 云 RTASR 服务进行字幕生成。
- **华为云 RASR**: 支持使用华为云 RASR 服务进行字幕生成。

## Downloads
